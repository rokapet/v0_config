# Voron V0.2 (V0.4240)
This is the Klipper config of my Voron V0.2 printer. Building is ~currently underway...~ finished, calibrating and tuning is in progress!

The printer is entirely self sourced. A few of the parts are repurposed from another printer that was torn down.

## Components
 - [BTT Pi v1.2](https://biqu.equipment/products/bigtreetech-btt-pi-v1-2) (and U2C v1.0 with CandleLight firmware, for CAN bus)
 - [BTT SKR Mini v2.0](https://biqu.equipment/products/bigtreetech-skr-mini-e3-v2-0-32-bit-control-board-integrated-tmc2209-uart-for-ender-4)
 - [LDO frame](https://shop.zen3d.eu/ldo-voron-v02-gray-frame) (grey)
 - [LDO 42STH48-2504AC Speedy Power](https://ldomotion.com/products/ldo42sth482504ac) (NEMA17) for XY
 - [Siboor 42STH26-0804A-200](https://www.aliexpress.com/item/1005005850132970.html) (TR8x8) for Z
 - [TriangleLab Sherpa Mini extruder with LDO 36STH20-1004AHG stepper](https://trianglelab.net/products/sherpa-mini-extruder?VariantsId=10515)
 - [CNA (RDBB) Z1 rails](https://www.aliexpress.com/item/32880801484.html)
 - [Mornsun LMF200-23B24 power supply](https://www.mornsun-power.com/index/sitesearch/partlink/keyword/LMF200-23B24.html)
 - [Haldis Red Lizard hotend](https://www.aliexpress.com/item/1005004527280702.html) with [Mellow 60W 6x15mm heater cartridge](https://3dmellow.com/products/mellow-1pcs-24v-60w-615mm-ceramic-cartridge-heater-heating-tube-for-3d]-printer-parts-e3dv6-j-head-phaetus-dragon-hotend) and [TriangleLab E-V6 Plated Copper nozzle](https://trianglelab.net/products/e-v6-plated-copper-nozzle?VariantsId=10536)
 - [Energetic PEI Pro flexible bed](https://www.aliexpress.com/item/1005005551646331.html)
 - GDSTime fans ([40x40x10 blower](https://www.aliexpress.com/item/32799248824.html) and [30x30x10 axial](https://www.aliexpress.com/item/32814252396.html))
